# MirrorKit v1.1 — Visual Customization Design Spec

**Date:** 2026-04-25
**Author:** Achraf Trabelsi
**Status:** Draft — pending implementation
**Target release:** MirrorKit v1.1 (free update for v1.0 buyers)

---

## 1. Goal

Add three independent visual customization features to MirrorKit, all shipping together as v1.1:

1. **Background expanded mode** — replace the hardcoded dark purple-blue gradient with a curated set of named presets (Midnight, Charcoal, Snow, Sunset, Ocean, Aurora, Black) plus a custom solid color picker.
2. **Bezel customizable** — replace the binary `showDeviceFrame` toggle with a style picker offering Classic (current colored bezel + Dynamic Island/notch, with black/silver/gold color choice), Floating (no bezel + soft drop shadow), and Frameless (raw video clipped to rounded corners).
3. **Annotations live** — let the user draw on top of the mirrored screen during demos using a small palette (pen, arrow, circle, rectangle, highlighter) with 6 color choices, undo, and clear all. Annotations are baked into snapshots (PNG output) but not into recordings (mp4 output).

This is a free update for v1.0 buyers. No StoreKit / IAP work.

## 2. Non-goals

- Photorealistic device mockup (Bezel option C from brainstorming) — rejected because Apple's marketing images aren't license-free and commissioning custom assets is a 1-2 week chantier of its own.
- Annotation persistence across sessions — annotations are always ephemeral, never written to disk.
- Annotations baked into recordings (Annotations option C from brainstorming) — rejected because per-frame composition is expensive and the user-value/effort ratio is much weaker than snapshot integration. Possible v1.2.
- Text annotation tool, eraser tool, stroke width slider — rejected as "Annotations option C". v1.1 ships the recommended 5-tool set only.
- Custom image / wallpaper for background (Background option C from brainstorming) — rejected for v1.1, possible v1.2.
- Image-based device mockups in Bezel — rejected (see above).
- Internationalization. MirrorKit ships English-only.

## 3. Product decisions (locked from brainstorming)

| Dimension | Decision |
|---|---|
| Background scope | Preset gradients + custom solid color (option B) |
| Background presets | 7 named: Midnight (default), Charcoal, Snow, Sunset, Ocean, Aurora, Black |
| Bezel scope | Classic (existing) + Floating (new) + Frameless (= existing showDeviceFrame=false) |
| Bezel default | Classic, color black |
| Annotations output | Live overlay + bakery into PNG snapshots, NOT recordings |
| Annotations UI | Extension of `FloatingToolbar` (2nd row visible only in annotation mode) |
| Annotations toolset | 5 tools (pen / arrow / circle / rectangle / highlighter) + 6 colors + undo + clear all |
| Annotations activation | Toggle button in toolbar + hotkey `A` |
| Annotations auto-clear | On device switch, on disconnect, on window close |
| Persistence migration | Legacy `showDeviceFrame` Bool → `bezelStyle` String, one-shot at first v1.1 launch |
| Monetization | Free update for v1.0 buyers — no StoreKit code |

### 4.2 Component boundaries

- **`BackgroundPreset`** (Models): pure value type. Enum `RawRepresentable<String>` with 8 cases (7 presets + `.custom`). Exposes `displayName: String` and `@ViewBuilder makeBackground(customColor: Color) -> some View`.
- **`BezelStyle`** (Models): pure value type. Enum `RawRepresentable<String>` with 3 cases (`.classic`, `.floating`, `.none`).
- **`Annotation`** (Models): pure value type. Struct with `id: UUID`, `tool: AnnotationTool`, `color: AnnotationColor`, `points: [CGPoint]` (normalized to [0,1]×[0,1] in video space).
- **`AnnotationCanvas`** (Core): `@Observable @MainActor final class`. Owns `annotations: [Annotation]`, `currentStroke: Annotation?`, `undoStack: [[Annotation]]`, `activeTool`, `activeColor`, `isAnnotationModeActive`. All mutations through methods (`beginStroke`, `extendStroke`, `endStroke`, `undo`, `clearAll`).
- **`AnnotationCompositor`** (Utils): namespace enum with one static function `compositeSync(frame: NSImage, annotations: [Annotation], canvasSize: CGSize) -> NSImage`. Pure with respect to inputs, sync.
- **`ColorPersistence`** (Utils): namespace enum with `Color(hex: String) -> Color?` and `hex(from: Color) -> String` helpers.

`DeviceManager` and `CaptureEngine` are **not modified** — these features live entirely in the UI/state layer.

### 4.3 Concurrency

- Everything stays on `@MainActor`: `AnnotationCanvas`, all SwiftUI views, all `@AppStorage` reads/writes.
- `AnnotationCompositor.compositeSync` is `nonisolated` — pure function called from the main thread (where snapshot is already taken).
- `CaptureEngine` remains an actor (unchanged).
- Conforms to `SWIFT_STRICT_CONCURRENCY = complete` (Swift 6.0).

Two `@AppStorage` keys:

- `backgroundPreset: String` — default `"midnight"`. Parsed via `BackgroundPreset(rawValue:) ?? .midnight`.
- `backgroundCustomColor: String` — default `"#1F1C40"` (~midnight purple). Hex string parsed via `Color(hex:)`. On parse failure, fallback to `.black`.

### 5.4 Settings UI — `BackgroundPresetPicker`

The 8th cell is "Custom…" with a hatched/striped fill and a `+` icon — clicking opens the macOS shared `NSColorPanel` and binds its color to `backgroundCustomColorHex`.

### 6.2 Persistence + legacy migration

Two new `@AppStorage` keys:

- `bezelStyle: String` — default `"classic"`.
- `bezelColor: String` — default `"black"`. Only applied when `style == .classic`.

**Migration** — extracted as a pure function for testability:

```swift
// Models/BezelStyle.swift (or AppDelegate)
enum LegacyMigration {
    /// Migrates v1.0's showDeviceFrame Bool to v1.1's bezelStyle String.
    /// Idempotent: only runs if bezelStyle is not yet set.
    /// Returns true if a migration was performed (for tests).
    @discardableResult
    static func migrateBezelStyleIfNeeded(in defaults: UserDefaults) -> Bool {
        guard defaults.object(forKey: "bezelStyle") == nil else { return false }
        guard let legacy = defaults.object(forKey: "showDeviceFrame") as? Bool else {
            defaults.set("classic", forKey: "bezelStyle")  // explicit init for new installs
            return true
        }
        defaults.set(legacy ? "classic" : "none", forKey: "bezelStyle")
        return true
    }
}
```

Called once in `AppDelegate.applicationDidFinishLaunching(_:)` or `MirrorKitApp.init`:

```swift
LegacyMigration.migrateBezelStyleIfNeeded(in: .standard)
```

### 6.3 `DeviceFrameView` rendering

Current `DeviceFrameView` always renders the colored bezel + island. New version branches at the top:

```swift
struct DeviceFrameView<Content: View>: View {
    let spec: DeviceFrameSpec
    let style: BezelStyle  // ← new parameter
    let content: Content

    var body: some View {
        switch style {
        case .classic:  classicBody    // existing implementation, refactored into private var
        case .floating: floatingBody   // new
        case .none:     framelessBody  // existing showDeviceFrame=false fallback
        }
    }

    private var floatingBody: some View {
        GeometryReader { geo in
            let baseline = min(geo.size.width, geo.size.height) / (spec.kind == .iPad ? 820 : 390)
            let outerRadius = spec.cornerRadius * baseline
            content
                .clipShape(RoundedRectangle(cornerRadius: outerRadius, style: .continuous))
                .shadow(color: .black.opacity(0.45), radius: 24, x: 0, y: 12)
        }
    }

    private var framelessBody: some View {
        content.clipShape(RoundedRectangle(cornerRadius: 44, style: .continuous))
    }
}
```

`floatingBody` reuses the same `outerRadius` calculation as `classic` (so the rounded corners match the device's actual radius), but skips the bezel rectangle, the Dynamic Island capsule, and the frame color — just shadow + content.

## 7. Annotations live design

### 7.1 Models

```swift
// Models/Annotation.swift
import SwiftUI
import CoreGraphics

enum AnnotationTool: String, CaseIterable {
    case pen, arrow, circle, rectangle, highlighter

    var iconName: String {
        switch self {
        case .pen:         return "pencil.tip"
        case .arrow:       return "arrow.up.right"
        case .circle:      return "circle"
        case .rectangle:   return "rectangle"
        case .highlighter: return "highlighter"
        }
    }

    var defaultStrokeWidth: CGFloat {
        switch self {
        case .pen, .circle, .rectangle: return 2
        case .arrow:                    return 3
        case .highlighter:              return 24
        }
    }
}

enum AnnotationColor: String, CaseIterable {
    case red, blue, green, yellow, white, black

    var color: Color {
        switch self {
        case .red:    return Color(red: 1.00, green: 0.23, blue: 0.19)
        case .blue:   return Color(red: 0.04, green: 0.52, blue: 1.00)
        case .green:  return Color(red: 0.20, green: 0.78, blue: 0.35)
        case .yellow: return Color(red: 1.00, green: 0.84, blue: 0.04)
        case .white:  return .white
        case .black:  return .black
        }
    }
}

struct Annotation: Identifiable, Equatable {
    let id: UUID
    let tool: AnnotationTool
    let color: AnnotationColor
    /// Coordinates normalized to [0,1]×[0,1] of the video display layer.
    /// Pen and highlighter use the full points array; arrow/circle/rectangle use [start, current].
    var points: [CGPoint]
}
```

### 7.2 `AnnotationCanvas`

```swift
@MainActor
@Observable
final class AnnotationCanvas {
    private(set) var annotations: [Annotation] = []
    private(set) var currentStroke: Annotation?

    var activeTool: AnnotationTool = .pen
    var activeColor: AnnotationColor = .red
    var isAnnotationModeActive: Bool = false {
        didSet {
            if !isAnnotationModeActive { currentStroke = nil }
        }
    }

    @ObservationIgnored private var undoStack: [[Annotation]] = []
    private static let maxUndoDepth = 20

    func beginStroke(at point: CGPoint) {
        currentStroke = Annotation(
            id: UUID(),
            tool: activeTool,
            color: activeColor,
            points: [point]
        )
    }

    func extendStroke(to point: CGPoint) {
        guard var stroke = currentStroke else { return }
        switch stroke.tool {
        case .pen, .highlighter:
            stroke.points.append(point)
        case .arrow, .circle, .rectangle:
            stroke.points = [stroke.points[0], point]
        }
        currentStroke = stroke
    }

    func endStroke() {
        guard let stroke = currentStroke else { return }
        // Discard zero-length strokes (single tap with no drag)
        guard stroke.points.count > 1 else {
            currentStroke = nil
            return
        }
        pushUndoSnapshot()
        annotations.append(stroke)
        currentStroke = nil
    }

    func undo() {
        guard let previous = undoStack.popLast() else { return }
        annotations = previous
    }

    func clearAll() {
        guard !annotations.isEmpty else { return }
        pushUndoSnapshot()
        annotations.removeAll()
    }

    private func pushUndoSnapshot() {
        undoStack.append(annotations)
        if undoStack.count > Self.maxUndoDepth {
            undoStack.removeFirst()
        }
    }
}
```

### 7.4 `AnnotationLayer`

```swift
struct AnnotationLayer: View {
    let canvas: AnnotationCanvas

    var body: some View {
        GeometryReader { geo in
            Canvas { context, size in
                for annotation in canvas.annotations {
                    drawAnnotation(annotation, context: context, size: size)
                }
                if let inflight = canvas.currentStroke {
                    drawAnnotation(inflight, context: context, size: size)
                }
            }
            .gesture(drawingGesture(in: geo.size))
        }
    }

    private func drawingGesture(in size: CGSize) -> some Gesture {
        DragGesture(minimumDistance: 0, coordinateSpace: .local)
            .onChanged { value in
                let normalized = CGPoint(
                    x: max(0, min(1, value.location.x / size.width)),
                    y: max(0, min(1, value.location.y / size.height))
                )
                if canvas.currentStroke == nil {
                    canvas.beginStroke(at: normalized)
                } else {
                    canvas.extendStroke(to: normalized)
                }
            }
            .onEnded { _ in canvas.endStroke() }
    }

    private func drawAnnotation(_ annotation: Annotation, context: GraphicsContext, size: CGSize) {
        AnnotationDrawing.draw(annotation, in: context, size: size)
    }
}
```

**Drawing extracted to a shared namespace** so the live layer (above) and the snapshot compositor (section 7.8) use the same code:

```swift
// UI/AnnotationLayer.swift (or its own file) — namespace enum
enum AnnotationDrawing {
    /// Renders a single annotation into a SwiftUI GraphicsContext.
    /// Coordinates in the annotation are normalized [0,1] and scaled by `size`.
    static func draw(_ annotation: Annotation, in context: GraphicsContext, size: CGSize) {
        let denormalized = annotation.points.map {
            CGPoint(x: $0.x * size.width, y: $0.y * size.height)
        }
        let strokeColor = annotation.color.color
        let lineWidth = annotation.tool.defaultStrokeWidth

        switch annotation.tool {
        case .pen:
            guard !denormalized.isEmpty else { return }
            var path = Path()
            path.move(to: denormalized[0])
            for p in denormalized.dropFirst() { path.addLine(to: p) }
            context.stroke(path, with: .color(strokeColor),
                           style: StrokeStyle(lineWidth: lineWidth, lineCap: .round, lineJoin: .round))

        case .highlighter:
            guard !denormalized.isEmpty else { return }
            var path = Path()
            path.move(to: denormalized[0])
            for p in denormalized.dropFirst() { path.addLine(to: p) }
            context.stroke(path, with: .color(strokeColor.opacity(0.35)),
                           style: StrokeStyle(lineWidth: lineWidth, lineCap: .round, lineJoin: .round))

        case .arrow:
            guard denormalized.count == 2 else { return }
            drawArrow(from: denormalized[0], to: denormalized[1], in: context, color: strokeColor, width: lineWidth)

        case .circle:
            guard denormalized.count == 2 else { return }
            let rect = boundingRect(denormalized[0], denormalized[1])
            context.stroke(Path(ellipseIn: rect), with: .color(strokeColor), lineWidth: lineWidth)

        case .rectangle:
            guard denormalized.count == 2 else { return }
            let rect = boundingRect(denormalized[0], denormalized[1])
            context.stroke(Path(roundedRect: rect, cornerRadius: 4), with: .color(strokeColor), lineWidth: lineWidth)
        }
    }

    private static func boundingRect(_ a: CGPoint, _ b: CGPoint) -> CGRect {
        CGRect(x: min(a.x, b.x), y: min(a.y, b.y),
               width: abs(b.x - a.x), height: abs(b.y - a.y))
    }

    private static func drawArrow(from start: CGPoint, to end: CGPoint, in context: GraphicsContext,
                                  color: Color, width: CGFloat) {
        var path = Path()
        path.move(to: start)
        path.addLine(to: end)
        // Arrowhead — two short lines at 30° on either side of the shaft.
        let angle = atan2(end.y - start.y, end.x - start.x)
        let headLength: CGFloat = 14
        let p1 = CGPoint(x: end.x - headLength * cos(angle - .pi / 6),
                         y: end.y - headLength * sin(angle - .pi / 6))
        let p2 = CGPoint(x: end.x - headLength * cos(angle + .pi / 6),
                         y: end.y - headLength * sin(angle + .pi / 6))
        path.addLine(to: p1)
        path.move(to: end)
        path.addLine(to: p2)
        context.stroke(path, with: .color(color),
                       style: StrokeStyle(lineWidth: width, lineCap: .round, lineJoin: .round))
    }
}
```

`AnnotationDrawing` lives alongside `AnnotationLayer.swift` (or in its own file `Utils/AnnotationDrawing.swift` if the team prefers — same module either way). The file layout in section 4.1 stays unchanged.

### 7.8 Snapshot composite

`takeSnapshot()` in `MirrorContentView.swift` (around line 262 of v1.0) is updated:

```swift
func takeSnapshot() {
    guard let pixelBuffer = displayLayer.lastPixelBuffer else { NSSound.beep(); return }

    let baseImage = SnapshotEncoder.makeNSImage(pixelBuffer: pixelBuffer)
    let composited = AnnotationCompositor.compositeSync(
        frame: baseImage,
        annotations: annotationCanvas.annotations,
        canvasSize: baseImage.size
    )

    guard let pngData = composited.pngData() else { NSSound.beep(); return }

    do {
        let url = try ExportManager.savePNG(pngData)
        print("[MirrorKit] Snapshot saved: \(url.path)")
        NSSound(named: "Grab")?.play()
        showToast("Snapshot saved — \(url.lastPathComponent)", revealing: url)
    } catch {
        print("[MirrorKit] Snapshot failed: \(error.localizedDescription)")
        NSSound.beep()
    }
}
```

`SnapshotEncoder` exposes a new helper `makeNSImage(pixelBuffer:) -> NSImage` (the existing `encodePNG` already does most of the work; we split it).

`AnnotationCompositor`:

```swift
enum AnnotationCompositor {
    @MainActor
    static func compositeSync(frame: NSImage, annotations: [Annotation], canvasSize: CGSize) -> NSImage {
        guard !annotations.isEmpty else { return frame }

        // Render the annotation layer to a CGImage at the frame's pixel resolution.
        let renderer = ImageRenderer(content:
            AnnotationLayerStaticRenderer(annotations: annotations)
                .frame(width: canvasSize.width, height: canvasSize.height)
        )
        renderer.scale = 1.0  // we're working at native pixel dimensions already
        guard let overlay = renderer.cgImage else { return frame }

        // Composite frame + overlay into a new NSImage.
        let result = NSImage(size: frame.size)
        result.lockFocus()
        defer { result.unlockFocus() }
        frame.draw(in: NSRect(origin: .zero, size: frame.size))
        if let context = NSGraphicsContext.current?.cgContext {
            let rect = CGRect(origin: .zero, size: frame.size)
            context.draw(overlay, in: rect)
        }
        return result
    }
}

/// Renders annotations only, no gestures, no canvas state — used by the snapshot compositor.
private struct AnnotationLayerStaticRenderer: View {
    let annotations: [Annotation]
    var body: some View {
        Canvas { context, size in
            for annotation in annotations {
                AnnotationDrawing.draw(annotation, in: context, size: size)
            }
        }
    }
}
```

`AnnotationLayer.draw(_:context:size:)` is extracted as a `static` helper (or moved to a `AnnotationDrawing` enum) so both the live layer and the snapshot compositor use the same rendering code. **Single source of truth** — what the user sees on screen is exactly what the PNG contains.

## 9. Testing strategy

### 9.1 New test target — `MirrorKitTests`

Bootstrap a Swift Testing target via `project.yml` (the project has zero tests today). Three test files cover the testable pure-logic pieces.

**`BackgroundPresetTests`** (~20 lines):
- `BackgroundPreset(rawValue: "midnight") == .midnight`
- `BackgroundPreset(rawValue: "xyz") == nil`
- All cases have distinct `displayName`s

**`BezelStyleTests`** (~25 lines, including migration):
- `BezelStyle(rawValue: "classic") == .classic`, idem for floating, none
- `LegacyMigration.migrateBezelStyleIfNeeded`:
  - Legacy `showDeviceFrame: true` → `bezelStyle: "classic"`
  - Legacy `showDeviceFrame: false` → `bezelStyle: "none"`
  - Already-set `bezelStyle` → no-op (idempotent)
  - No legacy and no current → defaults to `"classic"`
- `FrameColor` round-trip via raw value

**`AnnotationCanvasTests`** (~80 lines):
- Empty canvas has no annotations
- `endStroke()` commits a multi-point stroke; single-point stroke is discarded (no zero-length artefacts)
- `extendStroke()` on `.pen` appends; on `.circle` replaces endpoint
- `undo()` reverts last stroke; on empty stack is no-op
- Undo stack capped at 20 (the 21st snapshot evicts the oldest)
- `clearAll()` removes annotations and is itself undoable
- `isAnnotationModeActive = false` clears `currentStroke` (via `didSet`)

## 10. Risks and mitigations

| Risk | Likelihood | Impact | Mitigation |
|---|---|---|---|
| `ImageRenderer` SwiftUI Canvas snapshot blank or misaligned | Medium | Medium | Visual verification during Phase 4 Task 16; if it breaks, fallback to `Core Graphics` direct draw using the same shape math. |
| `Color(hex:)` parser fails on custom-color edge cases | Low | Low | Fallback to `.black` ; tests cover the parser. |
| Bare `A` hotkey conflicts with future text input | Low | Low | Documented constraint; reconsider if v1.2 adds text annotation. |
| Annotations look wrong after iPhone rotation (resolution change) | Medium | Medium | Auto-clear on resolution change via `.onChange(of: detectedResolution)` (covered in 7.3). Manual test. |
| Legacy migration overwrites a user's manually set `bezelStyle` on first v1.1 launch | Low | Medium | Migration is idempotent — only runs if `bezelStyle` is unset. Tests cover this. |
| Floating bezel shadow looks too heavy on Snow background | Low | Low | Manual visual review on each background; tweak `shadow opacity` if needed. |
| App Store reviewer flags the bare `A` hotkey as conflicting with system shortcuts | Very low | Low | macOS doesn't reserve bare letters for system actions. If flagged, rebind to `Cmd+Shift+A`. |
